# jiadaolibtv-drama-director

贾维斯导演 Agent，用于把短剧剧本、飞书剧本链接、已有分镜或 LibTV 提示词草稿，拆解成 LibTV 可执行的导演方案。

这个 skill 面向 AI 短剧生产流程，重点覆盖：

- 剧本理解与分镜拆解
- LibTV 视频生成提示词
- 角色、场景、道具一致性资产提示词
- 角色音色、音调、音线一致性
- LibTV 画布节点规划
- 成片质检与重做建议

## 适用场景

当你需要把短剧剧本落到 LibTV 生产流程时，可以使用这个 skill，例如：

```text
贾导，处理这个剧本：
贾导，拿这个飞书剧本做 LibTV 分镜：
贾导，把这个分镜转成 LibTV 提示词并建画布：
贾导，按这个剧本生成角色、场景、道具资产并写入 LibTV：
```

## 安装

把本仓库复制到你的 Codex skills 目录：

```bash
mkdir -p ~/.codex/skills
cp -R jiadaolibtv-drama-director ~/.codex/skills/
```

目录中必须保留 `SKILL.md`。

## 输出能力

默认完整交付包括：

1. 剧本读取状态
2. 剧本理解与拆解
3. 人物、场景、道具表
4. 角色音色音调音线表
5. 情绪曲线
6. LibTV 可生成性判断
7. 完整分镜脚本
8. 每个镜头的 LibTV 提示词
9. 角色三视图、表情图、动作图提示词
10. 场景、道具参考图提示词
11. LibTV 画布节点规划
12. 生成执行结果或待执行方案
13. 素材缺口清单
14. 成片质量质检
